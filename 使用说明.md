# 三星堆文化数字展示平台 - 使用说明

## 快速开始

### 第一步：解压文件

将 `qingshitree-3d-delivery.zip` 解压到任意文件夹

### 第二步：启动程序

**双击运行 `start.bat`**

- 会弹出黑色命令行窗口（**不要关闭**）
- 浏览器会自动打开 `http://localhost:8088`
- 如果浏览器没有自动打开，手动在地址栏输入：`http://localhost:8088`

### 第三步：等待加载

首次打开需要等待模型加载（约 10-30 秒），请耐心等待

### 第四步：关闭程序

使用完毕后，点击黑色命令行窗口，按 `Ctrl + C` 停止服务器

## 常见问题

### 1. 双击 `start.bat` 没反应？

**原因**：未安装 Python

**解决方法**：
- 访问 https://www.python.org/downloads/
- 下载并安装 Python 3.x
- 安装时勾选 "Add Python to PATH"

### 2. 浏览器显示"无法访问此网站"？

**原因**：服务器未启动或端口被占用

**解决方法**：
- 确认 `start.bat` 已运行（黑色命令行窗口存在）
- 如果 8088 端口被占用，编辑 `server-cors.py`，将 `PORT = 8088` 改为 `PORT = 8080`，然后重新运行 `start.bat`

### 3. 页面卡在"正在加载模型"？

**原因**：模型文件较大，首次加载需要时间

**解决方法**：
- 等待 30 秒左右
- 如果超过 1 分钟仍无反应，按 F12 查看控制台错误信息，并联系开发者

### 4. 模型显示不出来，报错 CORS？

**原因**：直接双击了 `index.html`，没有通过服务器访问

**解决方法**：
- **不要直接双击 `index.html`**
- 必须通过 `http://localhost:8088` 访问（运行 `start.bat` 后自动打开）

## 项目说明

本作品为三星堆文物 3D 数字展示平台，包含三件三星堆代表性文物的高精度 3D 模型：

1. **青铜神树** - 高 3.96 米，现存最大青铜树
2. **青铜立人像** - 通高 260.8 厘米，古蜀国巫师兼王者形象
3. **青铜人头像** - 敷有金面具，神秘庄重的祭祀形象

每件文物均配有交互式标注点，点击可查看详细解说。
