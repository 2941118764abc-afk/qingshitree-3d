# 三星堆文化数字展示平台

## 快速开始

1. 双击 `start.bat` 启动服务器
2. 浏览器自动打开 `http://localhost:8088`
3. 等待模型加载（约 10-30 秒）

## 操作说明

- **鼠标左键拖动** - 旋转模型
- **鼠标滚轮** - 缩放模型
- **鼠标右键拖动** - 平移视角
- **点击左侧菜单** - 切换文物
- **点击标注点** - 查看说明

## 详细文档

请查看 `使用说明.md` 获取完整使用指南和常见问题解答。

## 注意事项

- 必须通过 `http://localhost:8088` 访问，不能直接双击 `index.html`
- 关闭时按 `Ctrl+C` 停止服务器

## 技术栈

Three.js + Python http.server
